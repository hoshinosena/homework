使用Java重写的链表类LinkListClass
=====================
###改进与特性
- 删除了建立节点LinkNode时不必要的赋值
- head节点允许包含data
- tail节点支援
- 提供了point节点直接操纵链表而不必手动创建新的LinkNode对象
- 维护了size成员，也保留了size()方法。所有要调用size()方法的操作都将受益，大量节点时性能显著提高。
